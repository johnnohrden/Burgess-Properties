# Burgess Properties Website Rebuild

A ground-up rebuild of the website for Burgess Properties, LLC, a commercial real estate brokerage based in Massachusetts. Built and maintained by John Nohrden.

## Built With
- Vanilla HTML, CSS, and JavaScript
- Bootstrap (bundled locally)
- Developed using Claude Code

## Features
- No external dependencies at runtime
- AI-optimized SEO with llms.txt
- Responsive design across all pages
- Individual listing detail pages with images and brochures

## Status
Active development. Live site serving commercial real estate listings across Greater Boston.
